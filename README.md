# 股票分析工具

一个轻量级股票分析工具，支持命令行和网页界面：

- 在线拉取日线收盘价
- 计算 `SMA`、`EMA`、`RSI`、`MACD`
- 输出趋势判断、涨跌幅和波动率
- 生成价格走势图表
- 离线 `demo` 模式，便于本地直接试跑
- 网页界面，支持交互式分析

## 运行环境

- Python 3.10+
- 网页应用需要：Flask>=2.0, matplotlib>=3.0

## 安装依赖

```bash
pip install -r requirements.txt
```

## 快速开始

### 命令行使用

```bash
./stock aapl.us
```

不带股票代码默认走在线模式，会提示参数错误。

你也可以直接输入中文名称（会自动解析成代码）：

```bash
./stock 中际旭创
./stock 英维克
```

不带参数会进入交互模式，按提示输入即可：

```bash
./stock
```

如果当前环境无法访问网络，可以使用：

```bash
./stock --source demo
```

在线模式默认优先尝试 `stooq`。如果 `stooq` 返回空数据、验证码页或不可用，程序会自动回退到 Yahoo Finance 的公开行情接口继续获取数据。

如果你想先确认本地环境是否正常，可以运行内置自检：

```bash
./stock --self-test
```

如果在线请求卡住，可以缩短超时：

```bash
./stock 601318.cn --timeout 5
```

或者直接指定走 Yahoo：

```bash
./stock 601318.cn --source yahoo
```

### 网页应用

启动网页界面：

```bash
python3 stock_web.py
```

浏览器访问：

```text
http://127.0.0.1:5000
```

网页功能：
- 输入股票代码或中文名称
- 选择数据来源（auto/stooq/yahoo/demo）
- 调整参数（记录条数、超时时间）
- 可选生成价格图表
- 查看详细分析结果

## 股票代码格式

`stooq` 常见代码格式，也兼容在线回退时的常用写法：

- 美股：`aapl.us`
- 港股：`0700.hk`
- A 股：`600519.cn`

## 说明

- 默认数据源为 `stooq`，但在线抓取失败时会自动回退到 Yahoo Finance
- `demo` 模式下股票代码参数可以省略
- `--self-test` 会检查核心指标计算和示例输出
- 分析结论基于技术指标规则打分
- 输出仅供研究和学习，不构成投资建议
