# 股票分析工具
添加文字

一个轻量级命令行股票分析工具，支持：

- 在线拉取日线收盘价
- 计算 `SMA`、`EMA`、`RSI`、`MACD`
- 输出趋势判断、涨跌幅和波动率
- 离线 `demo` 模式，便于本地直接试跑

## 运行环境

- Python 3.10+

## 快速开始

```bash
python3 stock_analyzer.py aapl.us
```

不带股票代码默认走在线模式，会提示参数错误。

如果当前环境无法访问网络，可以使用：

```bash
python3 stock_analyzer.py --source demo
```

在线模式默认优先尝试 `stooq`。如果 `stooq` 返回空数据、验证码页或不可用，程序会自动回退到 Yahoo Finance 的公开行情接口继续获取数据。

如果你想先确认本地环境是否正常，可以运行内置自检：

```bash
python3 stock_analyzer.py --self-test
```

## 示例输出

```text
股票代码: aapl.us
数据区间: 2025-10-01 -> 2026-04-22
最新收盘价: 214.30

关键指标
- SMA20: 210.14
- SMA60: 201.22
...
```

## 股票代码格式

`stooq` 常见代码格式，也兼容在线回退时的常用写法：

- 美股：`aapl.us`
- 港股：`0700.hk`
- A 股：`600519.cn`

## 说明

- 默认数据源为 `stooq`，但在线抓取失败时会自动回退到 Yahoo Finance
- `demo` 模式下股票代码参数可以省略
- `--self-test` 会检查核心指标计算和示例输出
- 分析结论基于技术指标规则打分
- 输出仅供研究和学习，不构成投资建议
